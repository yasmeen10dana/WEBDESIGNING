Code Cheat Sheet
  • <!DocType>-identifies what kind of document is being loaded.<!DOCTYPE><html></html>
  • <a>-used to attach a link.<a>a href....</a>
  • <abbr>-defines what needs to be abbreviated.<abbr>"I Scream For Ice Cream">ISIC</abbr>
  • <article>-where you can enter blog entries, post, etc.<article><h2>Yahoo</h2><p>Yahoo is a browser....</p></article>
  • <audio>- used to insert audio into the document.<audio><source src="icecream.org"type="audio/org"></audio>
  • <b>-make items bold in webpage.<b>Words to bold</b>
  • <body>- where you can include information for the HTML document.<body>Webpage content inserted here....</body>
  • <canvas>- renders graphics for scripting language.<canvas id="myCanvas"></canvas>
  • <data>- used for script friendly data.<li><data value="57575"></data></li>
  • <div>- a divider in the webpage.<div style="background-color:blue"><h3>this is a heading</h3></div>
  • <form>- for user input.<form action="/action.page"</form>
  • <h1>-headline, title of page.<h1>This is heading</h1>
  • <h2>-subtitles of the page.<h2>This is heading</h2>
  • <h3>-level 3 heading.<h3>This is heading</h3>
  • <h4>-level 4 heading.<h4>This is heading</h4>
  • <h5>-level 5 heading.<h5>This is heading</h5>
  • <h6>-level 6 heading.<h6>This is heading</h6>
  • <head>-the heading section of a webpage.<head><title>Insert title here...</title></head>
  • <html>-all other tags go inside the HTML tag.<html><head><title>Ice Cream</title></head></html>
  • <i>-makes text italicized. <i>Different flavors</i>
  • <img>-used for displaying images.<img src="ice cream.jpg">
  • <input>-used for various attributes.<input type=text>
  • <li>-list item<li>Homework</li><li>Exams</li>
  • <link>- attached external websites to webpages.<link rel="stylesheet">
  • <ol>- ordered list.<ol><li>Food</li></ol>
  • <p>-paragraph.<p>Text in paragraph</p>
  • <script>-used for javascript applications in HTML.<script> <document.getElementById></script>
  • <span>-Applying styles around text.<span style="color:blue"> </span>
  • <strong>-makes text stronger.<strong>Strong text</strong>
  • <style>-declares style sheets in the document.<style>h1 {color:pink;}</style>
  • <sub>- for displaying subscript.<sub>subscript</sub>
  • <table>-makes tables in the HTML page.<table><tr><th>Sprinkles<th></table>
  • <td>-table data<td>Table entry</td>
  • <th>-table header<th>cell entry</th>
  • <tr>-makes a row in a table.<tr><th>table info<th></tr>
  • <u>-annotates texts.<u>paragraph</u>
  • <ul>-unordered lists.<ul><li>Food</li></ul>
  • <var>-defines a variable within the content.<var>variable</var>
  • <video>-embedding video into HTML file.<video width=320><source src="movie.mp3"></video>
